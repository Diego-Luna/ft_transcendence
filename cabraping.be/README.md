# Backend API Docs

| Method | Endpoint       | Description         |
| ------ | -------------- | ------------------- |
| GET    | /              |                     |
| GET    | /api           |                     |
| GET    | /api/users     |                     |
| POST   | /api/users     |                     |
| GET    | /api/games     | Get all games       |
| GET    | /api/games/:id | Get one game by id  |
| POST   | /api/games     | Create a new game   |
| DELETE | /api/games/:id | Delete a game by id |
